Java API for ISO 20022 standard

Tools20022 project provides Java API generated from the public eRepository.

The generated API contains the full repository, message definitions from all areas, business concepts and the metamodel.

Features

  • No external dependencies, just pure Java 8 code.
  • Typesafe implementation, no "STRING_CONSTANTS" required in applications.
  • Regularly updated (Usually within two weeks after a new eReposotory published on iso20022.org)
